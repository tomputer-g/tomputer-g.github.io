# Hello, QR World

This is a sample memo reachable at `tomgao.net/qr/hello`.

Print the QR code for this URL, stick it on something, and scanning it will bring you here.

## How to add a memo

1. Create a new `.md` file in `src/data/memos/`
2. Name it whatever ID you want (e.g. `desk.md` → `/qr/desk`)
3. Deploy the site

## Markdown is fully supported

- **Bold**, _italic_, `code`
- Lists, tables, headers
- Links and images
